# ui端到端自动化测试工具
* 基于puppeteer的UI测试工具

## 安装
```sh
npm install -g e2etest  # 正常安装
npm install -g e2etest --ignore-scripts # 避免chromium下载失败问题
```

## 主要功能
* [x] 提供测试Case执行及返回执行结果
* [x] 接口调用执行Case
* [x] 定时执行测试Case
* [x] 执行结果查看
* [x] 命令行启动
* [ ] chromium配置
* [ ] 测试文件存放地址默认改为用户路径地址+/.e2etest
* [ ] 删除过期测试文件
